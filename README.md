# 📚 Library Management System (Кітапхананы басқару жүйесі)

Бұл жоба — заманауи технологиялар көмегімен жасалған, кітаптарды іздеуге, броньдауға және басқаруға арналған онлайн платформа.

## 🚀 Жобаның негізгі идеясы

Пайдаланушылар кітаптар каталогын көре алады, аты немесе авторы бойынша іздеу жасайды және өздеріне қажетті кітапты таңдап, броньдай алады.

### 🔍 Негізгі мүмкіндіктер:

1.  **Кітаптарды іздеу:** Аты немесе авторы бойынша жылдам іздеу (мысалы: "Abai" деп іздесе, Абай Құнанбаевтың кітаптары шығады).
2.  **Толық ақпарат:** Әр кітаптың атауы, авторы, шыққан жылы, сипаттамасы және қазіргі статусы (қолжетімді немесе жоқ).
3.  **Броньдау жүйесі:** Егер кітап базада болса, пайдаланушы оны "Тапсырыс беру" батырмасы арқылы өз атына бекіте алады.
4.  **Жеке кабинет:** Пайдаланушы тіркеліп, өзі броньдаған кітаптар тізімін көре алады.
5.  **Админ панель:** Кітаптарды қосу, өшіру және кімнің қандай кітап алғанын бақылау мүмкіндігі.

---

## 🛠 Технологиялық стек

* **Frontend/Backend:** Next.js
* **Деректер базасы:** PostgreSQL
* **Стильдер:** CSS/Tailwind
* **Аутентификация:** JWT (JSON Web Token)

---

## ⚙️ Орнату және іске қосу нұсқаулығы

Жобаны өз компьютеріңізде іске қосу үшін төмендегі қадамдарды орындаңыз.

### 1. Жобаны көшіру (Clone)
Алдымен GitHub-тан жобаны көшіріп алыңыз:
```bash
git clone https://github.com/your-username/library-project.git
cd library-project
```

### 2. Қажетті бағдарламалар
Компьютеріңізде мыналар орнатылған болуы тиіс:
* **Node.js** (v18 немесе одан жоғары)
* **PostgreSQL** (Орнатылған және іске қосулы болуы керек)

### 3. .env файлын баптау
Жобаның негізгі папкасында `.env` файлын жасап, келесі деректерді енгізіңіз:

```env
DB_USER=postgres
DB_PASSWORD=password
DB_HOST=localhost
DB_PORT=5432
DB_NAME=library_db
JWT_SECRET=super_secret_key_123
```
*Ескерту: `DB_PASSWORD` бөлігіне өзіңіздің PostgreSQL пароліңізді жазыңыз.*

### 4. Пакеттерді орнату
```bash
npm install
```

### 5. Базаны дайындау және мәліметтерді толтыру
Базаға алғашқы админді және кітаптар тізімін қосу үшін келесі командаларды кезекпен іске қосыңыз:

```bash
# Админ аккаунтын жасау
node create-admin.js

# Кітаптар базасын толтыру (Seed)
node seed-books.js
```

**Админ деректері:**
* **Email:** admin@gmail.com
* **Password:** admin123

### 6. Жобаны іске қосу
```bash
npm run dev
```

Енді браузерден `http://localhost:3000` сілтемесіне өтіп, жобаны тексере аласыз.
